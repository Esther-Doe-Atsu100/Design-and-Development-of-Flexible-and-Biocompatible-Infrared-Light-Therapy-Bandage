# Design-and-Development-of-Flexible-and-Biocompatible-Infrared-Light-Therapy-Bandage
Flexible and biocompatible infrared (IR) light therapy bandage for wound healing using 850 nm LEDs, digital thermostat, and MOSFET timer relay. Final-year Biomedical Engineering project, All Nations University, Koforidua, Ghana. November 2025.
